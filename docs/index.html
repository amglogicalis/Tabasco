El puente definitivo entre tu PC y la Inteligencia Artificial

Una aplicación de escritorio de alto rendimiento que conecta tu workspace local a Gemini, ChatGPT y Claude. Gratis, sin API keys y con bypass anti-detección.

🤖

Sin API Keys

Olvídate de pagar por tokens. Tabasco aprovecha la interfaz web pública de tus bots para habilitar un backend local 100% gratuito.

🕵️

Bypass Anti-Detección

Con tecnología Stealth integrada. Emula huellas de usuario, tiempos de interacción aleatorios y pegado enriquecido de QuillJS para eludir protecciones automatizadas.

💻

Tabasco Code (Agente)

El núcleo de desarrollo. La IA puede leer tu estructura de archivos, escribir o reescribir código y lanzar comandos en tu terminal local de forma segura.

🔒

Sandbox de Seguridad

Seguridad de archivos absoluta. El agente opera encapsulado en el workspace asignado, protegiendo tus archivos del sistema contra accesos externos.

🔑

Perfiles Persistentes

Los datos y logins se guardan de forma aislada e independiente en tu máquina local. Inicia sesión una vez y chatea indefinidamente.

🚀

Interfaz Premium

Experiencia de usuario de alto nivel: burbujas redimensionables en altura, selector rápido de modelos, historial completo y consola integrada.

Guía de Instalación

Clona el repositorio e instala los binarios necesarios en menos de 2 minutos:

Terminal
# 1. Instala las dependencias
pip install -r requirements.txt

# 2. Descarga el motor Chromium
python -m playwright install chromium

# 3. Comprueba los imports del sistema
python check_imports.py

# 4. Lanza Tabasco
python main.py
